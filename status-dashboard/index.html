Fullbreidd rekstrarskrá yfir útgáfuval úr router: hvaða route-ástæða valdi mobile eða desktop, hvaða tæki/vafri fylgdi með og hversu margar taldar production heimsóknir liggja að baki.

Gæðaviðvaranir

Staðfest production vandamál eru sýnd sem viðvaranir. Óstaðfest, informational og diagnostic/test merki eru aðskilin og hækka ekki viðvörunarstöðu.

Persónuvernd: Engin nöfn, netföng, kennitölur, raw IP, raw user-agent texti eða Power BI innri gögn eru birt. Síðan birtir eingöngu aggregate/status gögn úr Google Apps Script endpointi, notar örugga HTML-escaping við birtingu og aðgreinir gateway, raw og diagnostic merki frá töldum heimsóknum.
