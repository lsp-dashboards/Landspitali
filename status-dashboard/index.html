Stjórnborð stöðu

Samsett rekstrarstig, tafarlausar athuganir og jafnvægi milli notkunar, leiðingar, uppruna og ferskleika.

Tracker patch

Staða nýjustu tracker uppfærslu, payload samnings, teljara-gates og aðgreiningar á gateway, diagnostic og töldum heimsóknum.

Gateway funnel

Root index views og dashboard-smellir eru gateway telemetry. Þau eru sýnd í eigin flæði svo þau ruglist ekki saman við taldar dashboard heimsóknir.

Signal panorama

Heatmap-yfirlit sem sýnir hvaða mælaborð, leiðingarástæður og upprunarásir bera notkunina. Breiðari reitir merkja meiri þunga í flæðinu.

Mælaborð

Staða eftir mælaborði. Tölurnar hér byggja eingöngu á töldum production heimsóknum úr aggregate/status gögnum.

Leiðing og uppruni

Greinir valda útgáfu, sjálfvirka eða þvingaða leiðingu og hvaðan töld umferð kemur.

Rekstrarinnsýn

Samstillt yfirlit fyrir notkun, leiðingu, gæði, tæknimerki og ferskleika. Hver reitur hefur eina aðaltölu og stutta rekstrartúlkun.

Tæki og vafraumhverfi

Toppsamsetningar, dreifingar og tæknileg merki. Ótalin eða diagnostic merki geta birst sem rekstrarathuganir en hækka ekki heimsóknir.

Skjár og aðgengi

Reported theme, forced-dark óvissa, viewport flokkar og aðgengismerki. Reported gildi eru aðskilin frá óvissum eða inferred merkjum.

Flutningur og afköst

Transport, payload stærð, cache og net-/tækjamerki þegar þau eru tiltæk. Compact diagnostic yfirlit án sjónræns hávaða.

Sjálfvirk leiðing

Ástæður og valin útgáfa þegar þær upplýsingar eru til. Debug/test/diagnostic og count_as_visit = FALSE línur eru síaðar frá.

Uppruni

Umferðarflokkur, UTM source og herferð úr samantekt. Root index er gateway telemetry og birtist í funnel, ekki sem talin heimsókn.

Birting á island.is

Opinberar kortaupplýsingar og birtingarstaða.

Gæðaviðvaranir

Staðfest production vandamál eru sýnd sem viðvaranir. Óstaðfest, informational og diagnostic/test merki eru aðskilin og hækka ekki viðvörunarstöðu.

Persónuvernd: Engin nöfn, netföng, kennitölur, raw IP, raw user-agent texti eða Power BI innri gögn eru birt. Síðan birtir eingöngu aggregate/status gögn úr Google Apps Script endpointi, notar örugga HTML-escaping við birtingu og aðgreinir gateway, raw, diagnostic og tracker-patch merki frá töldum heimsóknum.
